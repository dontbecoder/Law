## 第一章 基本原则

- 六大原则和其子原则

  1、合法行政

  ​	1.1、法律优先 = 必须遵守现行有效的法律

  ​	1.2、法律保留 = 应当依照法律授权活动，不得法外设定权力

  2、合理行政

  ​	2.1、公正、公平 = 同等情况同等对待，不同情况差别对待

  ​	2.2、考虑相关因素 = 只考虑与立法授权目的相关的各种因素，不考虑与授权目的不相关的因素

  ​	2.3、比例原则

  ​		2.3.1、合目的性 = 手段必须符合法律目的

  ​		2.3.2、适当性 = 手段必须能够达到目的或者至少有助于目的的达成

  ​		2.3.3、损害最小 = 有多种手段可供选择时，选择侵害相对人权益最小的

  3、程序正当

  ​	3.1、回避 = 任职回避和公务回避

  ​	3.2、公众参与

  ​		3.2.1、决定前 = 告知相对人

  ​		3.2.2、决定中 = 听取相对人陈述申辩或听证

  ​		3.2.3、决定后 = 行政机关说明理由，告知相对人依法享有复议或起诉的权利

  ​	3.3、行政公开 = 除涉及国家秘密和依法受到保护的商业秘密、个人隐私的外，应当公开

  4、诚实守信

  ​	4.1、信息真实 = 行政机关公布的信息应当全面、准确、真实

  ​	4.2、信赖保护

  ​		4.2.1、构成要件 = 授益行为 + 有信赖利益且该利益值得保护

  ​		4.2.2、保护方式 = 存续保护 + 财产保护

  5、高效便民

  ​	5.1、行政效率 = 应当遵守法定时限，积极履行法定职责

  ​	5.2、便利当事人 = 方便公民、法人和其他组织

  6、权责一致

  ​	6.1、行政效能 = 法律、法规应当赋予其实现目的所需的相应的执法手段

  ​	6.2、行政责任 = 行政机关违法或者不当行使职权，应当依法承担法律责任

1、目的。为控制与预防危险发生、防止证据灭失 VS 为实现其“基础决定”

## 第六章 行政程序

- 行政程序的概念和价值

  行政程序 = 行政主体实施行政行为时所应当遵循的方式、方法和步骤、时限和顺序的结合形式

  行政程序的功能和价值：

  1、完善沟通渠道，提高行政行为的社会可接受程度

  2、行政程序有利于行政机关全面获得行政资讯，使行政决定更加集思广益，提升行政决策品质

  3、行政程序体现了对公民人性尊严的尊重

  4、正当程序是限制公权力滥用的有效手段

  5、正当程序是维持社会秩序的稳定的有效手段

- 违反法定程序的案例解析

  1、未履行告知义务和未听取陈述和申辩

  ​	1.1、未履行告知义务

  ​	1.2、未听取陈述和申辩

  ​	1.3、未表明执法身份和执法的合法性

  ​	1.4、未有效告知

  2、未经法定程序 = 指法律、法规或者规章中规定了必经行政程序，而行政机关未履行程序义务作出的行政行为（eg，应当听证而未听证；未立案作出行政决定，未委托鉴定，未经重大决策程序，未经催告，未送达行政决定等）

  3、程序步骤或次序混乱 = 指启动行政程序后，行政机关未依法定的程序步骤或者次序履行程序义务（eg,出现程序颠倒，程序重叠等违法情形）

  4、程序方式不合法 = 行政机关履行程序义务时应当按照法定的程序方式进行，否则会导致程序违法

  5、超过法定期限 = 在行政程序中，一般规定了受理、决定、鉴定、送达等方面的期限要求。超过法定期限是否构成程序瑕疵，需要根据是否对权利构成实际影响、有无延期等来加以判断

## 第七章 政府信息公开

- 政府信息公开制度概述

  政府信息公开 = 指公民、组织对行政机关在行使行政职权的过程中掌握或控制的信息拥有**知情权**，除法律明确规定不予公开事项外，行政机关应当通过有效方式向公众和当事人公开。

  政府信息 = 指行政机关在履行行政管理职能过程中制作或获取的，以一定形式记录、保存的信息。

- 政府信息的公开主体

  1、行政机关 = 行政机关和法律、法规授权的具有管理公共事务职能的组织。

  ​	实行“谁制作，谁公开；谁保存，谁公开”的原则，法律、法规另有规定的，从其规定。

  ​	行政机关制作的信息，由制作机关负责公开；

  ​	行政机关从公民、法人和其他组织获取的信息，由保存该信息的机关负责公开；

  ​	行政机关获取的其他行政机关政府信息，由制作或最初获取该信息的机关负责公开

  2、派出机构、内设机构

  依照法律、法规对外以自己名义履行行政管理职能的，可以由该机构负责公开与所履行行政管理职能有关的政府信息公开工作

  3、共同制作的信息

  两个以上行政机关共同制作的，由**牵头制作的行政机关**负责公开。牵头机关在收到公开申请后可以征求相关机关的意见，被征求意见机关应当自收到征求意见书之日起15个工作日内提出意见，逾期未提出意见的视为同意公开。

- 政府信息公开的范围 = 以公开为原则，以法定不公开为例外（涉及国家秘密、商业秘密、个人隐私的，所公开的信息不得危及国家、公共、经济安全和社会稳定）

  1、涉及国家秘密一律不公开

  2、涉及商业秘密、个人隐私原则上不公开

  2.1、实体规则 = 涉及商业秘密、个人隐私的，原则上不公开，但以下情形除外：1）经权利人同意公开的；2）行政机关认为不公开可能对公共利益造成重大影响的

  2.2、程序规则 = 依申请公开的政府信息，公开会损害第三方合法权益的，应**书面征求**第三方意见；第三方应当自收到征求意见书之日起**15**个工作日内提出意见。在经过征求意见后，行政机关可以分情况处理：

  ​	1）第三方逾期未提出意见的，由行政机关依照本条例的规定决定

  ​	2）第三方不同意公开且有合理理由的，不予公开

  ​	3）行政机关认为不公开可能对公共利益造成重大影响的，可以决定公开，并将决定公开的内容和理由书面告知第三方

  ​	4）内部信息不公开 = 行政机关内部事务信息（人事管理、后勤管理、内部工作流程等），可以不公开

  3、过程性信息不公开 = 行政机关在履行行政管理职能过程中形成的讨论记录、过程稿、磋商信函、请示报告等，可以不予公开。法律、法规、规章规定上述信息应当公开的，从其规定。

  4、公开审查机制

  ​	行政机关应当建立健全政府信息公开审查机制，明确审查的程序和责任（依照《保守国家秘密法》以及其他法律、法规和国家有关规定对拟公开的政府信息进行审查）。

  ​	行政机关不能确定是否可以公开的，应当依照法律、法规和国家有关规定报有关主管部门或者保密行政管理部门确定。

- 政府信息公开的程序

  1、主动公开的范围 = 对涉及公众利益调整、需要公众广泛知晓或需要公众参与决策的政府信息，行政机关应当主动公开

  2、主动公开的方式和场所

  2.1、方式 = 建立健全信息发布机制，通过政府公报、政府网站或其他政务媒体、新闻发布会、报刊、广播、电视等途径予以公开。各级政府应当在国家档案馆、公共图书馆、政务服务场所设置查阅场所，并配备相应设备，提供便利。

  2.2、场所

  ​	法定公开场所：**各级国家档案馆**/**公共图书馆**/政务服务场所

  ​	选择公开场所：公共查阅室、资料索取点、信息公告栏、电子信息屏等

  3、主动公开程序

  3.1、公开信息目录和公开指南

  3.2、公开信息文本本身

  ​	3.2.1、期限：应当在信息形成或变更之日起20个工作日内决定是否公开，另有规定的除外

  ​	3.2.2、保密审查：对拟公开信息应当进行保密审查。**不能确定是否公开时，应当报有关主管部门或同级保密工作部门确定**。

  4、依申请公开政府信息的程序

  4.1、申请（可以口头，无需说明申请信息的用途）

  4.2、答复

  ​	4.2.1、起点

  ​			申请人当面提交申请的，以提交之日为收到申请之日

  ​			以邮寄方式提交的，以签收之日为收到之日；以平信等无需签收方式邮寄的，以收到之日为准

  ​			以互联网或传真提交的，以双方确认之日为收到之日

  ​	4.2.2、终点

  ​			当场答复 = 能当场就当场

  ​			非当场答复 = 应当自收到申请之日起**20个工作日**内，需要延长答复期限的，应当经负责人同意并告知申请人，最长不超过**20个工作日**

  ​	4.2.3、答复方式

  ​		1）申请内容不明确的，应当给予指导和释明，并自收到申请之日起7个工作日内一次性告知申请人作出补正，说明需要补正的事项和合理的补正期限。答复期限自行政机关收到补正的申请之日起计算。申请人无正当理由逾期不补正的，视为放弃，行政机关不再处理该政府信息公开申请。

  5、防止滥用申请权的制度规定

  6、申请公开和主动公开转化

- 信息更正程序

  申请人有证据证明行政机关提供的与其自身相关的政府信息记录不准确的，可以要求行政机关更正。

- 监督和保障

  1、举报

  2、提起行政争讼 = 相对人认为行政机关在政府信息公开工作中的**具体行政行为**（不公开、公开信息涉及个人秘密、拒绝更正错误信息和公开形式不符合要求等）侵犯其合法权益的，可以依法申请行政复议或提起行政诉讼


## 第八章 被告和复议被申请人

- 委托的被告 = 委托机关是被告

- 行政授权的被告 = （先看是否有授权资格，再看授权是否有效）有授权的，授权机关是被告；无授权的，视为委托，以委托机关为被告

- 共同行为的被告 = 原告只告一个，**通知原告追加被告**；原告不同意的，法院**应当追加没被起诉的为第三人**

- 假共同行为的被告 = 只能以其中的行政主体作为被告，非行政主体应当被列为第三人；法院应当对两个独立的被诉行政行为**分别立案**，但**可以合并审理**（法官拥有自由裁量权，只有**法官认为可以合并且当事人同意合并的**才能合并）

- 统一许可的被告 = “统一许可”即统一办理，又称为并联式审批，指某一许可事项涉及多个部门的职权，由某个行政机关作为主办机关统一受理申请，再转告其他部门分别提出意见后统一办理。应以“**实质不利影响机关**”为被告。告错的情况下，1）法院通知原告变更被告；2）原告拒绝变更的，裁定驳回起诉

- 经批准行为的被告资格的确认 = 行政诉讼采用的是**“看名义”的形式标准**；对经批准行为，行政复议采纳的是实质标准，复议被申请人是行政级别比较高的上级行政机关

- 开发区被告资格问题

  1、**国务院、省级政府批准设立的开发区管理机构**作出的行政行为，以该开发区管理机构为被告

  2、国务院、省级政府批准设立的开发区管理机构**所属职能部门**作出的行政行为，以该职能部门为被告

  3、对于处罚权，取得**法律、法规的授权**的机构，可以成为被告

  4、对于处罚权，**规章**无权让临时机构、企事业单位等非行政主体获得行政主体资格，不能成为被告

- 行政机关共同制作的信息公开

  1、两个以上行政机关共同制作的政府信息，由牵头制作的行政机关负责公开

  2、应当**以对外发生法律效力的文书上署名的**机关为被告

  3、所申请公开的信息已经主动公开的，告知申请人获取该政府信息的方式、途径，而不应以此为理由，直接拒绝公开

## 第九章 原告、复议申请人和第三人

- 原告 = 指**认为行政行为侵犯其合法权益**，而依法向法院提起诉讼的公民、法人或其他组织

  1、只要当事人“认为”受到行政行为的侵犯就可以起诉，这是当事人的一种主观认识，而不一定是其权利在实体法上确实受到了影响

  2、“必然影响”既包括权利被剥夺，也包括必然限制

  3、原告包括行政相对人（还包括行政相关人），只要是与行政行为由法律上直接利害关系的公民、法人或其他组织，对该行政行为不服的，均可作为原告起诉

  4、**债权人**一般不能直接以非债务关系当事人的行政机关为义务对象主张权利，**除非**相关法律规范已经明确规定 --- 行政机关负有考量和保护债务关系的法定职责或者行政机关的履责行为已经直接对债务关系产生法律拘束力

  5、**中外合资企业、中外合作企业、联营企业**的合资、合作、联营各方，认为**企业权益**或者**自己一方**的合法权益受到行政行为侵害的，可**以自己的名义**起诉

- 第三人 = 公民、法人或者其他组织**同被诉行政行为有利害关系但没有提起诉讼**，或者**同案件处理结果有利害关系的**，可以作为第三人申请参加诉讼，或者由法院通知参加诉讼

- **投诉、举报人的原告资格问题 *****

  起诉人无论是基于公益或其他意图就城建、治安管理等领域的违法违规行为提出举报或投诉后，即便发生不予查处或者查处不力等现象，其也**无权**就此向法院提起行政诉讼（但举报人或投诉人与违法行为存在特定法律关系，如相邻关系、共有关系等情况下，享有诉权，具有原告资格）

  【知识拓展】投诉举报人原告资格的实务观点

  判断第三人是否具有原告资格，即在于根据相关行政法律规范，行政机关在履行相关法定职责时，除了维护行政管理秩序外，是否同时还负有直接保护特定利害关系人合法权益的义务

  1、在原告主体资格上，一般仍限于提起自益形式的公益诉讼，仍然坚持原告本人需要提供证据证明其存在与普通公众不同的独特的权益，且该种权益受行政实体法律规范所保护，并存在被诉行政行为侵害的可能性

  2、若原告要求被告依法催违法行为进行查处，而自身合法权益并未受到违法行为侵害时，原告并不具有原告资格

- **股份制企业的原告资格**  **

  在行政法中，法定代表人、董事会、股东（大）会认为行政机关作出的行政行为侵犯企业经营自主权的，可以企业名义提起诉讼

## 第十章 行政诉讼和行政复议的管辖

- 要点提示

  1、一般地域管辖。行政案件由最初作出行政行为的行政机关所在地法院管辖。一般地域管辖采取“原告就被告”原则

  2、不动产案件的特殊地域管辖。由不动产所在地法院专属管辖

  3、经复议案件的特殊地域管辖。为了便利原告起诉，只要经过复议的行政案件，无论复议结果是维持或改变，原告既可以向原机关所在地法院起诉，也可以向复议机关所在地法院起诉

  4、限制人身自由案件的特殊地域管辖。被关了的人诉关了的行政强制措施，原告所在地（户籍所在地、经常居住地和被限制人身自由地）和被告所在地法院均有管辖权

## 第十一章 行政诉讼和行政复议的受案范围

具体行政行为 = 是行政主体**为了管理外部的公共事务**，针对外部对象而作出的行为（外部性的特征使具体行政行为与内部行政行为区分开来）

内部行政行为 = 是行政主体**为了管理内部事务**，对其内部组织或个人实施的行为：

1、针对组织内部行为：

​	1.1、上级行政机关决定下属机构的设立、增加、减少、合并

​	1.2、对行政组织内部权力的划定、调整

​	1.3、行政机关内部公文的流转

2、针对公务员的内部行为：奖惩、任免、培训、辞职等（最好用的判断方法为“**内外看身份**”）
